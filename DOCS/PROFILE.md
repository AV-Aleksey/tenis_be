# Профиль игрока

## Формат данных профиля

Базовый формат карточки:

- 🙋‍♂️ **Имя + ник**: `Алексей Freefire`
- 🖼️ **Аватар**: `https://.../avatar.jpg`
- 🇷🇺 **Локация**: `Москва: запад`
- 🏋️ **Уровень**: `Любитель`
- 🟢 **Статус**: `1`
- 📝 **Описание**: `Ищу регулярные матчи по вечерам`
- 💪 **Сила**: `1290`
- 🎾 **Матчи**: `Сыграл 0 матчей`
- ☏ **Телефон**: `+79279155591` (в интерфейсе показывается как скрытый)
- ✉️ **Telegram**: `@alekseyaav`

## Поля и правила

| Поле | Статус | Примечание |
|------|--------|------------|
| Имя + ник | Обязательно | Основной идентификатор в карточке |
| Аватар | Опционально | URL изображения профиля |
| Локация (город: район) | Обязательно | Используется для локального подбора |
| Уровень | Обязательно | Текстовое значение (например, «Любитель») |
| Статус | Опционально | Идентификатор статуса профиля (`status_id`) |
| Описание | Опционально | Короткое описание игрока |
| Сила | Обязательно | Основной числовой рейтинг |
| Кол-во сыгранных матчей | Системное | Считается автоматически |
| Телефон | Обязательно | Хранится, но показывается скрыто |
| Telegram | Обязательно | Основной контактный канал |

## Видимость

- Для других пользователей карточка доступна в режиме просмотра.
- Телефон всегда отображается в скрытом виде.
- Telegram отображается явно как основной способ связи.
- Сила и матчи — системные показатели, вручную не редактируются.

## Редактирование

- Игрок может редактировать: имя + ник, аватар, локацию, уровень, описание, телефон, Telegram.
- Системные поля (`Сила`, `Сыграл N матчей`) обновляются только системой.
